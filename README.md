# Preescoludi

Plataforma educativa interactiva para niños de preescolar y primero de primaria. Inspirada en experiencias lúdicas, visuales y seguras para el aprendizaje mediante el juego.

## Estructura del proyecto

```
/assets
   /images
   /audio
   /icons
/css
   style.css
   animations.css
   responsive.css
/js
   main.js
   owl.js
index.html
```

## Tecnologías
- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## Características
- Navbar infantil y amigable
- Hero principal colorido
- Búho animado con audio y microanimación
- Tarjetas educativas para temas clave
- Decoraciones flotantes SVG
- Diseño responsivo y modular
- Código limpio y fácil de escalar

## Personalización
- Reemplaza los archivos en `/assets/audio` y `/assets/icons` por tus propios recursos si lo deseas.
- El búho utiliza un SVG animado y un audio de bienvenida (`owl-welcome.mp3`).
